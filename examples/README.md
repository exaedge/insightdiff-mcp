# InsightDiff MCP Examples

These examples use the production MCP endpoint:

```text
https://insightdiff.exaedge.ai/mcp
```

Files:

- `remote-client-config.json`: remote MCP client config with API-key header.
- `prompt-examples.md`: common visual QA prompts for agents.
- `skill-install-commands.md`: copy-paste skill install commands.
- `mcp-inspector-smoke-test.md`: manual smoke-test checklist.

Never commit real `pv_live_...` keys. Replace `<YOUR_API_KEY>` only in your
private MCP client config.
